A collection of **Type-Safe, SSR-Compatible React Hooks** organized by category. Install any hook with a single command.

---

## Storage

Hooks for persisting data in the browser.

| Hook                                                            | Description                               |
| --------------------------------------------------------------- | ----------------------------------------- |
| [useLocalStorageState](/docs/hooks/use-local-storage-state)     | Sync state with localStorage (persistent) |
| [useSessionStorageState](/docs/hooks/use-session-storage-state) | Sync state with sessionStorage (per-tab)  |
| [useIndexedDB](/docs/hooks/use-indexed-db)                      | Store large data in IndexedDB             |
| [useFileSystem](/docs/hooks/use-file-system)                    | Read/write files to local disk            |

<MdxAccordions type="single">
<MdxAccordion title="Storage Comparison">

| Feature           | `useLocalStorageState` | `useSessionStorageState` | `useIndexedDB`       | `useFileSystem`   |
| ----------------- | ---------------------- | ------------------------ | -------------------- | ----------------- |
| **Storage Limit** | ~5MB per origin        | ~5MB per origin          | GBs (quota-based)    | Disk capacity     |
| **Data Types**    | JSON serializable      | JSON serializable        | Any (objects, blobs) | Files/Blobs       |
| **Persistence**   | Forever                | Until tab closes         | Forever              | Forever (manual)  |
| **Cross-tab**     | Syncs                  | Isolated                 | Shared               | N/A               |
| **API Style**     | Synchronous            | Synchronous              | Asynchronous         | Asynchronous      |
| **Best For**      | Preferences, tokens    | Forms, wizards           | Large data, offline  | Documents/editors |

</MdxAccordion>
</MdxAccordions>

## WebRTC

Hooks for building real-time communication applications.

| Hook                                                 | Description                                 |
| ---------------------------------------------------- | ------------------------------------------- |
| [useUserMedia](/docs/hooks/use-user-media)           | Access camera and microphone                |
| [usePeerConnection](/docs/hooks/use-peer-connection) | Manage WebRTC peer connections              |
| [useTrackToggle](/docs/hooks/use-track-toggle)       | Mute/unmute audio and video tracks          |
| [useScreenShare](/docs/hooks/use-screen-share)       | Share screen, window, or browser tab        |
| [useAudioLevel](/docs/hooks/use-audio-level)         | Detect audio levels for speaking indicators |
| [useDataChannel](/docs/hooks/use-data-channel)       | Send/receive data over peer connections     |

<MdxAccordions type="single">
<MdxAccordion title="WebRTC Hook Comparison">

| Hook                | Purpose                  | Key Returns                     |
| ------------------- | ------------------------ | ------------------------------- |
| `useUserMedia`      | Get local camera/mic     | `stream`, `start`, `stop`       |
| `usePeerConnection` | WebRTC signaling         | `createOffer`, `addTrack`, etc. |
| `useTrackToggle`    | Mute controls            | `toggleAudio`, `toggleVideo`    |
| `useScreenShare`    | Screen capture           | `stream`, `isSharing`           |
| `useAudioLevel`     | Voice activity detection | `level`, `isSpeaking`           |
| `useDataChannel`    | P2P messaging            | `send`, `messages`              |

</MdxAccordion>
</MdxAccordions>
