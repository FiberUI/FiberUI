A collection of type-safe, SSR-compatible React hooks.

## Available Hooks

| Hook                                                            | Description                               |
| --------------------------------------------------------------- | ----------------------------------------- |
| [useLocalStorageState](/docs/hooks/use-local-storage-state)     | Sync state with localStorage              |
| [useSessionStorageState](/docs/hooks/use-session-storage-state) | Sync state with sessionStorage            |
| [useIsMounted](/docs/hooks/use-is-mounted)                      | Detect if component is mounted (SSR safe) |
